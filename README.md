# Discord Information

This site is a simple Bootstrap-based information page for CougarTech's Discord community. It provides a place for frequently asked questions, tutorial videos, and general guidance for students and families who need quick information about using Discord.

## Maintainer

Maintained by Larry Roth.

## Deployment

This project is set up on Netlify and will automatically build and deploy whenever a new commit is pushed to the `main` branch.
